chevron_right {% trans "How the system recognizes your data" %}

{% trans "CSV files" %}: {% blocktrans trimmed %} Prefer clear filenames, for example locations.csv, tags.csv, links.csv. The name should contain the word location, tag, or link (any language casing). If the name is unclear, the first row is inspected: columns latitude + longitude mean locations; menu_name means tags; location_1 + location_2 mean links. {% endblocktrans %}

{% trans "Excel workbooks (.xlsx)" %}: {% blocktrans trimmed %} The workbook may have at most three worksheets. Each sheet must be for a different type: one for locations, one for tags, and/or one for links. Rename the sheet tabs so each name includes the right keyword—location, tag, or link—for example Locations, Tags, and Links. If a name contains several keywords, location is chosen before tag, and tag before link. Row 1 of each sheet must be the column headers, using the same column names as in the tables below (comma-separated CSV rules apply). {% endblocktrans %}
