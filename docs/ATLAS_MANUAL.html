ATLAS: Automated Topology LAnguage with Slicing

User Manual:

This manual will help you get started with the language and understand all of its commands.

Commands:

ADD_NODE (Alias: NODE, ADDNODE)

Syntax: ADD_NODE node_name xPos yPos [tag]

This command adds a node to the topology.

Example: ADD_NODE Node1 0 0

Example: NODE Node2 10 10 tag1

ADD_LINK (Alias: LINK, ADDLINK)

Syntax: ADD_LINK srcNode tgtNode [--color -weight -- opacity --dashed-line --dhased-line2 --dotted-line --rectlinear --srcAnchor --tgtAnchor]

This command adds a link between two nodes in the topology.

Example: ADD_LINK Node1 Node2 --color=red -weight=2

Example: LINK Node3 Node4 --opacity=0.5 --dashed-line

PUSH_POPUP (Alias: PUSHPOPUP)

Syntax: PUSH_POPUP nodeName content

This command pushs content in the popup of a specified node in the topology.

Example: PUSH_POPUP Node1 "This is the content of the popup"

Example: PUSHPOPUP Node2 "This is the content of the second popup"

SET_ICON (Alias: ICON)

Syntax: SET_ICON nodeName iconSource [flags]

This command sets an icon for a node in the topology.

Example: SET_ICON Node1 Icon1.jpg

Example: ICON Node2 --default-icon --size=small

ADD_BUTTON (Alias: BUTTON)

Syntax: ADD_BUTTON button_name x y commands [buttonLegend] [flags]

This command adds a button

Example: ADD_BUTTON Button1 0 0 "alert('Hello World!')"

Example: BUTTON Window2 Button2 10 10 "alert('Goodbye World!')" --color=red

RM_POPUP (Alias: RMPOPUP)

Syntax: RM_POPUP window_name

This command removes the popup of a node.

Example: RM_POPUP node1

ADD_SMART_LINK (Alias: SMART_LINK)

Syntax: ADD_SMART_LINK srcNode tgtNode [--color -weight -- opacity --dashed-line --dhased-line2 --dotted-line --rectlinear --srcAnchor --tgtAnchor]

This command adds a smart link between two nodes in the topology. This is a special type of link that can be used to create relationships between nodes.

Example: ADD_SMART_LINK Node1 Node2 --color=red -weight=2

Example: SMART_LINK Node3 Node4 --opacity=0.5 --dashed-line

HIGHLIGHT (Alias: HIGHLIGHT)

Syntax: HIGHLIGHT elementType elementName

This command highlights a specified element in the topology. This will change the color of the element making it more visible.

Example: HIGHLIGHT node Node1

Example: HIGHLIGHT link Link2

FADE (Alias: FADE)

Syntax: FADE elementType elementName

This command fades a specified element in the topology. This will change the color of the element making it less visible.

Example: FADE node Node1

Example: FADE link Link2
